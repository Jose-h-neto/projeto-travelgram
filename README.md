# Travel Profile

Projeto front-end desenvolvido com HTML e CSS que simula o perfil de um usuário em uma rede social, inspirado em layouts semelhantes ao Instagram.

## Visão geral

Este projeto representa o perfil de uma persona fictícia apaixonada por viagens, exibindo uma galeria de fotos e informações do usuário de forma organizada e visual.

O foco está na construção de layout e na apresentação de conteúdo, simulando uma interface real de rede social.

## Objetivos

- Reforçar fundamentos de HTML semântico  
- Praticar construção de layouts com CSS  
- Simular interfaces reais (UI)  
- Melhorar a organização de código front-end

## Funcionalidades

- Header com identidade visual e navegação  
- Seção de perfil com informações do usuário  
- Galeria de imagens em formato de grid  
- Layout inspirado em redes sociais  
- Footer simples  

## Tecnologias utilizadas

- HTML5  
- CSS3  

## Organização do CSS

O projeto utiliza separação de estilos por responsabilidade:

- `global.css`: estilos base
- `header.css`: cabeçalho  
- `nav.css`: navegação  
- `main.css`: área principal e galeria  
- `footer.css`: rodapé  
- `index.css`: importação eorganização dos módulos

## Como executar

1. Abra o arquivo `index.html` no navegador  

ou  

2. Utilize um servidor local (recomendado)  
   - Ex: Live Server no VS Code
